\doxysubsection{Detailed Description}
Contains all the functions related to networking. 


\begin{DoxyItemize}
\item Functions for binding to port, connecting to port and finding port.
\item Functions for sending, receiving and transmitting files.
\item Functions for sending and receiving a large buffer in small packets. 
\end{DoxyItemize}
